## Commands

- `cdk deploy Ec2SessionStack` : IpV4 Version.
- `cdk deploy Ec2OnPubSessionStack` : IpV4 Version, instance on public.
- `cdk deploy IpV6Ec2SessionStack` : IpV6 version.
